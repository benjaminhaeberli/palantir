## Oracle MJ

Inspirez-vous du *GM Crucible* de *Grimwild* : lancez 2 mots depuis une table à thèmes (atmosphère, lieu, situation), puis combinez-les et interprétez.

> *Exemple :*
> - *Qu'y a-t-il dans le temple ?* → `aged + beacon` → un artefact oublié.
> - *Pourquoi le village est-il si silencieux ?* → `tense + scheme` → une rébellion en préparation.
> - *Qu'est-ce qui a causé le carnage ?* → `flickering + rift` → un portail démoniaque.

Vous pouvez générer un oracle spécifique à votre univers via ChatGPT.
